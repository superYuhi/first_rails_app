# README

* MuscleGram

  URL:https://first-rails-app-yuhi.herokuapp.com
  新規投稿・いいね・ログイン・新規ユーザー登録などの機能を持つSNSです
  MuscleGramという名前は筋トレが好きなので命名しました

  SNS with functions such as new posting, likes, login, new user registration, etc.
  I named MuscleGram because I like muscle training.


* System dependencies
  Ruby 2.7.0
  Rails 6.0.2.1


* Usage
  ゲスト状態では投稿一覧のみを閲覧出来ますが、ユーザー登録をすると投稿詳細・ユーザー一覧・プロフィール等を閲覧できます
  新規投稿・投稿編集・プロフィール編集・投稿へのいいね等はログイン状態でのみ可能です
  パスワードの暗号化・アクセス制限等のセキュリティ対策済み

  In the guest, you can only see the post list, but if you login, you can see the post details, user list, profile, etc.
  New posts, post edits, profile edits, likes to posts, etc. are only possible when logged in.
  Security measures such as password encryption and access restrictions.
